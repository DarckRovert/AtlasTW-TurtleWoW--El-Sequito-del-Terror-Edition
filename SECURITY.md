# Security Policy — Atlas Navigator (Atlas-TW) 🔒🗺️

Protegemos la integridad de tu base de datos de mazmorras en **Turtle WoW**.

## 🛡️ Soporte por Versión
| Versión | Estado |
| --- | --- |
| v9.4.x | ✅ Soportada |
| v9.3.x | ⚠️ Crítico |

## 🛡️ Reporte de Vulnerabilidades
Si descubres un bug que compromise la estabilidad de las vistas predeterminadas de Atlas o que cause caídas de FPS inexplicables:

1.  **NO publique el exploit masivamente.**
2.  Reporta vía **Discord oficial** de **DarckRovert**.
3.  Incluye el log de `Atlas-TW -> TWLoot`.

## 🛡️ Protecciones Integradas
- **Loot Throttling**: El motor de carga de botín procesa los datos bajo demanda para evitar el consumo excesivo de memoria del cliente.
- **Database Isolation**: Aislamiento de la lógica de mazmorras custom para que no afecte el núcleo original de Atlas.
