# Code of Conduct — Atlas Navigator (Atlas-TW) 🛡️🗺️

Como miembros de **El Séquito del Terror** y colaboradores de **DarckRovert**, nos comprometemos a mantener una cartografía técnica y profesional.

## ⚖️ Nuestra Conducta
- **Rigor Cartográfico**: Cada tabla de botín debe estar verificada con el loot real de Turtle WoW.
- **Transparencia**: Todo cambio en las coordenadas de los jefes o misiones debe ser documentado en `CHANGELOG.md`.
- **Excelencia**: Priorizamos la precisión de la base de datos sobre cualquier ornato visual excesivo.

## 🚫 Comportamientos Inaceptables
- Introducir datos de botín falsos o de servidores privados no relacionados con Turtle WoW.
- Eludir la vinculación asíncrona con pfQuest.
- Alterar la identidad institucional del Séquito.

## 🛑 Reportes
Cualquier incidente debe reportarse directamente vía **Twitch** o **Discord** de **DarckRovert**.
